# 构造代码块

概述：写在一个类的成员位置的代码块

作用：可以把多个构造方法中，重复的内容抽离出来，放到构造代码块中

执行时机：优先于构造方法的执行，先执行构造代码块，再执行构造函数





示例：

```java
package test;

public class Student {
    private String name;
    private int age;
    
    // 把公共内容抽离出来放到构造代码块中
    {
        System.out.println("构造方法开始执行");
    }

    public Student() {
        // System.out.println("构造方法开始执行");
    }

    public Student(String name, int age) {
        // System.out.println("构造方法开始执行");
        this.name = name;
        this.age = age;
    }

    /**
     * 获取
     * @return name
     */
    public String getName() {
        return name;
    }

    /**
     * 设置
     * @param name
     */
    public void setName(String name) {
        this.name = name;
    }

    /**
     * 获取
     * @return age
     */
    public int getAge() {
        return age;
    }

    /**
     * 设置
     * @param age
     */
    public void setAge(int age) {
        this.age = age;
    }

    public String toString() {
        return "Student{name = " + name + ", age = " + age + "}";
    }
}

```
