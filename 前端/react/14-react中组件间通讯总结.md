# react中组件间通讯

## 组件间的关系

组件间基本上就是这三个关系：

* 父子组件
* 兄弟组件（非嵌套组件）
* 祖孙组件（跨级组件）



## 集中通讯方式

父——>子：props属性

子——>父：通过props传递函数，子调用父传递的函数，并传递形参

```
	1.props：
		(1).children props
		(2).render props
	2.消息订阅-发布：
		pubs-sub、event等等
	3.集中式管理：
		redux、dva等等
	4.conText:
		生产者-消费者模式
```



## 搭配方式

```
	父子组件：props
	兄弟组件：消息订阅-发布、集中式管理
	祖孙组件(跨级组件)：消息订阅-发布、集中式管理、conText(开发用的少，封装插件用的多)
```
