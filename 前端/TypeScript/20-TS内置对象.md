# TS内置对象

TS内置了一些对象，这些对象也内置了对应的类型。



示例：

```tsx

// 正则
const regexp: RegExp = /\w\d\s/


// 日期对象
const date:Date = new Date()


// 错误对象
const error:Error = new Error()

// 还有一些基础类型的
const num: Number = new Number();
const str: String = new String();
```

## BOM类型

而像bom的中的点击事件中的event，它也是有一个类型的

示例：

```tsx
document.addEventListener('click', function(e:MouseEvent) {
  console.log('TS的e',e);
})
```



## Promise

promise类型，我们需要给它指定泛型

示例：

```tsx
// 这里要指定泛型
function promise():Promise<number> {
  return  new Promise<number>((resolve, reject)=> {
    resolve(1)
  })
}

promise().then((res)=> {
  console.log(res);
})
```



## 怎么去找核心类型定义

当你在使用一些常用的方法的时候，TypeScript 实际上已经帮你做了很多类型判断的工作了

而他们的定义文件，则在 [TypeScript 核心库的定义文件](https://github.com/Microsoft/TypeScript/tree/master/src/lib)中

目录:`src/lib`
