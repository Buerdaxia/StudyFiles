# layout中xs，sm，md...

每行总共24个栅格，在不同尺寸的页面上如何分配宽度比例：

| 名称 | 尺寸    | 常用设备                |
| :--- | :------ | :---------------------- |
| xs   | <768px  | 超小屏 如：手机         |
| sm   | ≥768px  | 小屏幕 如：平板         |
| md   | ≥992px  | 中等屏幕 如：桌面显示器 |
| lg   | ≥1200px | 大屏幕 如：大桌面显示器 |
| xl   | ≥1920px | 2k屏等                  |



请配合element-ui上的layout组件一起查看。
