# box-decoration-break属性

作用：**`box-decoration-break`** 属性用来定义当元素跨多行、多列或多页时，元素的片段应如何呈现。

注意：该属性是实验性功能，需要适配

示例：

```css
-webkit-box-decoration-break: splice; /*截断*/
box-decoration-break: splice;

-webkit-box-decoration-break: clone; /*复制一份，就是跨行后，两行的样式一样*/
box-decoration-break: clone; 
```
