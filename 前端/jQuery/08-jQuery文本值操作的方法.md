# jQuery文本值操作的方法

操作文本相关的方法jQuery也封装了一些



## html方法

该方法和原生的innerHTML一模一样，都是可以给元素设置子元素（并且标签也能够设置）

语法：

```js
// 设置
$('div').html('<p>我是p标签</p>');
// 获取，不传参数就是获取
$('div').html()
```





## text方法

该方法和原生innerText方法也一模一样，只能设置字符串，会将传入的值全部当作字符，设置到对应元素的内容中

语法：

```js
// 设置
$('div').text('<p>我是p标签</p>');

// 获取
$('div').text()
```





##  val方法

它又和原生JS的value属性差不多，都是能够获取value值的

语法：

```js
// 设置
$('input').val('请输入内容');

// 获取
$('input').val()
```
