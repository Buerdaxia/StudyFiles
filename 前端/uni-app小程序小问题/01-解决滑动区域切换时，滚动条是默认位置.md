# 解决滑动区域切换时，滚动条是默认位置



一般：我们设置滑动区域是使用`<scroll-view>`标签



实现：我们切换，滑动时，先让滚动条，每次都是在最上面

核心：使用`:scroll-top`属性，每次切换时，都将该属性的值重置到最顶部即可。



小问题：就是`:scroll-top`不能够赋值重复的值，否则会没有效果，所以我们不能直接赋值为0，需要简单判断一下。



`html`部分：

```html
<scroll-view scroll-y="true" :style="{height: wh + 'px'}" :scroll-top="scrollTop">
</scroll-view>
```



`js`部分：

```js
 export default {
    data() {
      return {
        // 当前设备可用高度
        wh: 0,
        cateList: [],
        // 激活的一级分类
        active: 0,
        // 二级分类列表
        cateLevel2: [],
        // 滚动条距离顶部的值
        scrollTop: 0
      };
    },
   watch: {
     active(index) {
       // 动态获取二级分类数据
       this.cateLevel2 = this.cateList[index].children;
       // 切换时，重置scrollTop的高度，如果是0就赋值1，如果是1就赋值0，解决赋值不生效问题
       this.scrollTop === 0 ? this.scrollTop = 1 : this.scrollTop = 0;
     }
   }
}
```
